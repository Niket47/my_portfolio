# Blenheim Bedrooms - Case Study

## Overview

Blenheim Bedrooms is an e-commerce platform specializing in bedroom furniture. I worked as part of an 8-member team to completely redevelop the website and admin dashboard, improving functionality, UI/UX, and workflow efficiency.

## Project Scope

Complete redevelopment of:
- Customer-facing website
- Admin dashboard
- Order management system
- Inventory management
- Customer relationship management

## Technical Stack

- **Frontend**: React.js
- **Backend**: Node.js
- **Database**: MongoDB
- **Admin Panel**: Custom React dashboard

## Key Features Implemented

### Admin Dashboard
- Real-time order tracking
- Inventory management
- Customer management
- Analytics and reporting
- Product catalog management

### Website Improvements
- Enhanced product browsing
- Improved checkout process
- Better mobile responsiveness
- SEO optimizations

## Team Collaboration

Worked collaboratively with:
- Frontend developers
- Backend developers
- UI/UX designers
- QA testers
- Project managers

## Results

- Improved workflow efficiency
- Enhanced user experience
- Better admin tool functionality
- Increased platform stability
